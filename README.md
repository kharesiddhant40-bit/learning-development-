Smart Incident Management System

A modern Incident Management System built using Next.js that helps organizations manage, track, and resolve incidents efficiently.
This platform allows users to create incidents, track their progress, and enables administrators to manage users and assign engineers.

🚀 Features

User Authentication (Login / Register)

Create and Track Incidents

Role-based Dashboard

Admin Panel for User Management

Incident Status Tracking

Modern Responsive UI

🧑‍💻 Tech Stack

Frontend: Next.js, React

Language: TypeScript

Styling: Tailwind CSS

UI Components: Radix UI

Deployment: Vercel
